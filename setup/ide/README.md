# IDE設定

## 概要

プロジェクトや言語に依存しない、汎用的なIDEの設定を管理する。
個人の好みに依存する設定（テーマ、フォント、キーバインド等）は含めず、チームや環境を問わず共通で使える設定のみを対象とする。

## 対象IDE

- **VSCode** / **Cursor** — `settings.json` 形式で提供
- IntelliJ系 — 将来的に対応を検討

VSCodeとCursorは同じ `settings.json` 形式のため、共通の設定ファイルで対応する。

## 含まれる設定

| カテゴリ       | 内容                                                                      |
| -------------- | ------------------------------------------------------------------------- |
| ファイル       | エンコーディング(UTF-8)、改行コード(LF)、末尾空白の除去、最終行の改行挿入 |
| エディタ       | タブ幅、スペース挿入、保存時フォーマット、括弧の自動閉じ                  |
| エクスプローラ | ファイルネスティング（関連ファイルのグループ化）                          |
| Git            | スマートコミット候補の無効化、同期確認の無効化                            |
| JS/TS          | 自動インポート、インポートパスの形式                                      |

## 含めない設定

- テーマ、アイコン、カラースキーム
- フォント、フォントサイズ
- キーバインド
- ウィンドウレイアウト（タイトルバー、アクティビティバー等）
- 言語別フォーマッター（プロジェクトのテンプレート側で設定する）
- 特定の拡張機能に依存する設定

## 使い方

`settings.json` の内容を、自身のIDEのユーザー設定にマージして使用する。

# IDE設定

## 概要

プロジェクトや言語に依存しない、汎用的なIDEの設定を管理する。
個人の好みに依存する設定（フォント、キーバインド等）は含めず、チームや環境を問わず共通で使える設定のみを対象とする。

## 対象IDE

- **Cursor** — `cursor/settings.json`
- **VSCode** — `vscode/settings.json`

基本設定は共通で、VSCodeのみ GitHub Copilot の設定を追加している。

## 含まれる設定

| カテゴリ       | 内容                                                                      |
| -------------- | ------------------------------------------------------------------------- |
| ファイル       | エンコーディング(UTF-8)、改行コード(LF)、末尾空白の除去、最終行の改行挿入 |
| エディタ       | タブ幅、スペース挿入、保存時フォーマット、括弧の自動閉じ                  |
| エクスプローラ | ファイルネスティング（関連ファイルのグループ化）                          |
| Git            | スマートコミット候補の無効化、同期確認の無効化                            |
| JS/TS          | 自動インポート、インポートパスの形式                                      |

## 含めない設定

- フォント、フォントサイズ
- キーバインド
- ウィンドウレイアウト（タイトルバー等）
- 言語別フォーマッター（プロジェクトのテンプレート側で設定する）
- 特定の拡張機能に依存する設定（Copilotを除く）

## 使い方

各IDEのユーザー設定に対応する `settings.json` の内容をマージして使用する。
